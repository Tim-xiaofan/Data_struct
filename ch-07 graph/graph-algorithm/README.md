# kosaraju 算法
    求有向图的强连通分量，时间复杂度同遍历
# prime 算法
    求网的最小生成树，O(n ** 2), 边数无关，因此适用于求边稠密的网的最小生成树
# kruskal
    另一途径求网的最小生成树, 时间复杂度为O(eloge)
# find_articul 算法
    寻找连通图的关节点: O(n + e)
# dijkstra 算法
    用Dijkstra算法求有向网G的v0顶点到其余顶点v的最短路径P[v]及其带权长度D[v]
# floyd 算法
    用Floyd算法求有向网G中各对顶点v和w之间的最短路径P[v][w]及其带权长度D[v][w]
